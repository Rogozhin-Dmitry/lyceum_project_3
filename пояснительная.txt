Название: Adventure of Kyo
Авторы: Рогожин Дмитрий, Алёна Ващенко, Тимофей Ляхов	Авторы: Рогожин Дмитрий, Алёна Ващенко, Тимофей Ляхов
Описание идеи:
    Полноценный сайт для изучения иностранных языков (Английский, Немецкий), возможность проходить открытые тесты,
    узнавать свои рекорды, создавать собственные открытые и закрытые тесты.
Описание реализации:
    проект реализован в 17 основных файлах, с помощью 16 классов:
        Category, CreatedTest, MailingLists, TemporaryFirstTest, TemporaryFirstTestCreate, TemporarySecondTest,
        TemporarySecondTestCreate, TemporaryThirdTest, TemporaryThirdTestCreate, Test, User - Основные классы для
        создания таблиц в базе данных с помощью sqlalchemy, там описаны все поля и зависимости между таблицами, все они
        отнаследованны от класса SqlAlchemyBase

        TestCreateForm, RegisterForm, ChangeForm, SmallLoginForm - Основные классы для более простой работы с формами с
        помощью flask_wtf и wtforms, там описаны все поля форм а так-же принимаемые ими значения, все они отнаследованны
        от класса FlaskForm

    Сам функционал, все обработчики реализованны в файлах, main.py и extend_main_index_page.py, там содержится 32
    осовных обрботчика:
        logout, registration, login, edit_user, profile - Все обработчики необходимые для реализации личных кабинетов,
        входа, выхода, регистраии и тд.
        page_not_found, internal_error, info, contacts - Все одностроничные обработчики, контакты, сообщения об ошибках,
        информация и прочее.
        index, my_tests, open_user_tests, test_site - Все обработчики страниц выбора тестов, а так-же самой оснавной
        страницы теста
        test_create, third_test_create, third_test, first_test_create, first_test, second_test_create, second_test -
        Все страинцы создания и прохождения тестов.
        email, delete_test - Служебные обработчики, не имеющие своих страниц, занимаются только перенаправлением

Описание технологий:
    WEB фреймворк flask
    Библиотека для удобной работы с формой Flask-WTF
    Библиотека для удобной работы с пользователем Flask-Login
    Библиотека для работы с графикой PIL
    Библиотеки для работы с базами данных json, SQLAlchemy
    Библиотека для работы с несколькими потоками threading
    Библиотека для работы с файлами (копирование) shutil
    Библиотека для работы с рандомом random
    Системные библиотеки os
Дальнейшие пути развития:
    Добавление новых типов тестов,
    Добавление новых языков,
    Привязка к почте
